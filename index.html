AI 이미지 분류 프로젝트

과일 이미지 분류기

웹캠과 이미지 업로드 기능을 통해 과일 사진을 분석하고, 어떤 과일인지 예측 결과를 보여주는 AI 웹앱입니다.

프로젝트 소개

이 웹앱은 Teachable Machine으로 학습한 이미지 분류 모델을 활용하여 사용자가 웹캠 또는 업로드한 이미지를 분석할 수 있도록 만든 서비스입니다.
  • 웹캠으로 실시간 분류 가능
  • 이미지 업로드로 정적 사진 분류 가능
  • 예측 결과를 확률과 함께 시각적으로 확인 가능
사용 방법
1. 웹캠 시작 버튼을 눌러 실시간으로 분류합니다.
2. 또는 이미지 업로드 버튼으로 사진을 넣어 분석합니다.
3. 오른쪽 결과 영역에서 가장 높은 예측값과 전체 확률을 확인합니다.
